Python Frontmatter
=====================

.. module:: frontmatter

`Jekyll <http://jekyllrb.com/>`__-style YAML front matter offers a
useful way to add arbitrary, structured metadata to text documents,
regardless of type.

This is a small package to load and parse files (or just text) with YAML
front matter.
